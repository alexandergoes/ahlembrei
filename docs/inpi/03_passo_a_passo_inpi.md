# Passo a Passo — Registro de Programa de Computador (RPC) no INPI

## Pré-requisitos

- [ ] Conta **gov.br** nível **Prata** ou **Ouro**
  - Prata: validação via banco credenciado (Itaú, Santander, etc.)
  - Ouro: validação via certificado digital ou reconhecimento facial
  - Crie em: https://sso.acesso.gov.br
- [ ] CPF regularizado na Receita Federal
- [ ] Código-fonte completo em arquivo (.zip ou pasta)
- [ ] Documento descritivo do programa (já gerado neste diretório)
- [ ] Cartão de crédito ou boleto para GRU (R$ ~120 para pessoa física)

---

## Etapa 1 — Acessar o e-INPI

1. Acesse https://www.gov.br/inpi
2. Clique em **"Registrar Programa de Computador"**
3. Faça login com sua conta **gov.br**
4. Clique em **"Peticionar"**

## Etapa 3 — Anexar Documentos

O sistema solicitará upload de arquivos. Prepare:

1. **Código-fonte**: pasta completa em .zip (~2-5 MB)
2. **Documento descritivo** (adaptado deste .md para .pdf):
   - Pode ser um PDF gerado a partir deste arquivo
   - Inclua: título, versão, linguagem, finalidade, fluxo, funcionalidades
3. **Extrato do código**: primeiras e últimas 10 páginas (veja o arquivo `02_extracao_codigo_fonte.md`)
4. **Comprovante de pagamento da GRU**

---

## Etapa 4 — Pagamento (GRU)

1. O sistema gerará uma **GRU** (Guia de Recolhimento da União)
2. Valor estimado para pessoa física: **R$ 120,00**
3. Pague via **boleto bancário** ou **PIX**
4. O pagamento leva até 2 dias úteis para compensar

---

## Etapa 5 — Acompanhamento

1. Após o protocolo, o INPI analisa em **3 a 6 meses**
2. Acompanhe em: https://www.gov.br/inpi → Serviços → Acompanhar Petição
3. Se houver exigência (falta de documento), você será notificado por e-mail
4. Prazo para cumprir exigência: **30 dias**

---

## Dicas Importantes

- **Salve o número do protocolo** assim que finalizar o pedido
- **Tire print** de cada tela do formulário preenchido
- O código-fonte **não precisa ser registrado em cartório** — o INPI aceita o arquivo digital
- O registro vale por **50 anos** a partir da data de criação
- Se quiser acelerar, contrate um despachante (R$ 300-500) — mas é perfeitamente fazível sozinho

## Depois do Registro RPC

Com o RPC emitido, você pode:

1. **Solicitar Patente de Invenção** (PI) — proteção do conceito de negócio (12-24 meses de análise)
2. **Licenciar a tecnologia** para terceiros (hospitais, planos de saúde)
3. **Abrir processo de franquia** com marca registrada e software protegido
4. **Vender o software** com transferência de titularidade no INPI
