## 2. Definições

- **Titular**: usuário cadastrado que cria e mantém um perfil de emergência
- **Socorrista**: terceiro que acessa a página pública de emergência do Titular
- **Dados de Emergência**: dados médicos, contatos, documentos e localização cadastrados pelo Titular
- **QR Code**: código bidimensional gerado pela Plataforma que permite acesso ao perfil de emergência
- **@apelido**: identificador público único opcional associado ao perfil

## 3. Serviço

A Plataforma permite ao Titular:
- Cadastrar dados médicos, contatos de emergência, documentos e endereço
- Gerar QR Code físico para acesso aos dados de emergência
- Permitir que socorristas acessem dados críticos em situações de emergência
- Notificar automaticamente contatos cadastrados via WhatsApp
